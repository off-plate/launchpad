## Add a new link

Edit `index.html`. Each card is a `<a class="card" href="...">` block. Copy an existing one, swap title/desc/icon/URL. Commit. Netlify auto-deploys.

## Stack

Single static `index.html`. No build, no dependencies. Hosted on Netlify.
